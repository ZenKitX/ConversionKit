# ConversionKit v0.3.0 开发计划

## 版本目标

实现 UI 组件库，提供开箱即用的换算界面组件。

## 核心功能

### 1. 基础 UI 组件

#### CategoryGrid - 类别网格组件
显示所有换算类别的网格视图。

**功能:**
- 网格布局展示所有类别
- 每个类别显示图标、名称和描述
- 支持点击选择类别
- 响应式布局（适配不同屏幕尺寸）
- 支持自定义主题

**API 设计:**
```dart
CategoryGrid({
  required List<ConversionCategory> categories,
  required Function(ConversionCategory) onCategorySelected,
  int crossAxisCount = 3,
  double spacing = 16.0,
  CategoryGridStyle? style,
})
```

#### UnitSelector - 单位选择器
选择源单位和目标单位的组件。

**功能:**
- 下拉选择或底部弹窗选择
- 显示单位名称和符号
- 支持搜索过滤
- 支持最近使用和收藏
- 交换按钮（快速交换源和目标单位）

**API 设计:**
```dart
UnitSelector({
  required List<ConversionUnit> units,
  required ConversionUnit? selectedUnit,
  required Function(ConversionUnit) onUnitSelected,
  String? label,
  UnitSelectorStyle? style,
})
```

#### ValueInput - 数值输入组件
输入要转换的数值。

**功能:**
- 数字键盘输入
- 支持小数点
- 支持科学计数法
- 实时验证
- 清除按钮
- 单位显示

**API 设计:**
```dart
ValueInput({
  required TextEditingController controller,
  required ConversionUnit? unit,
  Function(String)? onChanged,
  String? label,
  ValueInputStyle? style,
})
```

#### ResultDisplay - 结果展示组件
显示转换结果。

**功能:**
- 大字体显示结果
- 单位符号显示
- 格式化显示（千分位）
- 复制按钮
- 动画效果

**API 设计:**
```dart
ResultDisplay({
  required double value,
  required ConversionUnit unit,
  bool showCopyButton = true,
  ResultDisplayStyle? style,
})
```

### 2. 复合组件

#### ConversionScreen - 完整换算页面
集成所有基础组件的完整换算界面。

**功能:**
- 类别选择
- 单位选择（源和目标）
- 数值输入
- 结果显示
- 历史记录
- 收藏功能

**API 设计:**
```dart
ConversionScreen({
  ConversionCategory? initialCategory,
  ConversionScreenStyle? style,
})
```

#### CurrencyScreen - 汇率换算页面
专门的汇率换算界面。

**功能:**
- 货币选择
- 金额输入
- 实时汇率显示
- 批量转换
- 汇率历史图表（可选）

**API 设计:**
```dart
CurrencyScreen({
  CurrencyConverter? converter,
  CurrencyScreenStyle? style,
})
```

#### MortgageScreen - 房贷计算页面
房贷计算界面。

**功能:**
- 贷款参数输入
- 还款方式选择
- 结果展示
- 还款计划表
- 图表展示（可选）

**API 设计:**
```dart
MortgageScreen({
  MortgageCalculator? calculator,
  MortgageScreenStyle? style,
})
```

### 3. 功能组件

#### HistoryList - 历史记录列表
显示最近的换算记录。

**功能:**
- 列表展示历史记录
- 点击快速重复换算
- 删除记录
- 清空所有记录
- 持久化存储

**API 设计:**
```dart
HistoryList({
  required List<ConversionHistory> history,
  Function(ConversionHistory)? onHistorySelected,
  Function(ConversionHistory)? onHistoryDeleted,
  HistoryListStyle? style,
})
```

#### FavoritesList - 收藏列表
显示收藏的换算组合。

**功能:**
- 列表展示收藏
- 点击快速使用
- 删除收藏
- 编辑收藏名称
- 持久化存储

**API 设计:**
```dart
FavoritesList({
  required List<ConversionFavorite> favorites,
  Function(ConversionFavorite)? onFavoriteSelected,
  Function(ConversionFavorite)? onFavoriteDeleted,
  FavoritesListStyle? style,
})
```

## 技术实现

### 1. 状态管理

使用 Provider 或 Riverpod 进行状态管理：
- 当前选择的类别
- 当前选择的单位
- 输入值和结果
- 历史记录
- 收藏列表

### 2. 数据持久化

使用 shared_preferences 存储：
- 历史记录（最近 50 条）
- 收藏列表
- 用户偏好设置

### 3. 主题系统

支持自定义主题：
- 亮色主题
- 暗色主题
- 自定义颜色
- 字体大小调整

### 4. 响应式设计

适配不同屏幕尺寸：
- 手机（竖屏/横屏）
- 平板
- 桌面
